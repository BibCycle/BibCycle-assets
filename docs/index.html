「BibCycle 翼旅」是一款以離線優先為核心、基於 Expo/React Native 建構的讀經 App。

關於 「BibCycle 翼旅」

「BibCycle 翼旅」是一款以「每日抽取經文章節」為核心的讀經 App。 你可以在安靜時刻抽取經文、記錄閱讀進度、收藏經文，並在登錄後同步你的記錄與設置。
