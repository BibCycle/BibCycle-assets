「BibCycle 翼旅」是一款以離線優先為核心、基於 Expo/React Native 建構的讀經 App。 "BibCycle Wing Journey" is an offline-first Scripture reading app built with Expo and React Native.

關於「BibCycle 翼旅」 About "BibCycle Wing Journey"

「BibCycle 翼旅」是一款以「每日抽取經文章節」為核心的讀經 App。 BibCycle Wing Journey is a Scripture reading app centered on drawing a daily passage. 你可以在安靜時刻抽取經文、記錄閱讀進度、收藏經文，並在登錄後同步你的記錄與設置。 You can draw passages in quiet moments, track reading progress, save favorites, and sync your records and settings after signing in.
