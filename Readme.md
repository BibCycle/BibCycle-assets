## 關於 BibCycle

- 以章節（Chapter）為最小閱讀單位
- 從未讀池中隨機抽取，每次只讀一章
- 沒有進度表，沒有落後感，停下來再回來，池子還在那裡
- 完成全部 1,189 章即完成一個 Cycle（週期）

BibCycle 不是一個叫你每天讀聖經的 App。它是一個讓你在停下來很久之後，仍然願意回來的地方。
